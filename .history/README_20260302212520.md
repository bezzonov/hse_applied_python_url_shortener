# HSE Applied Python 3 - URL Shortener

## Реализованные методы
- POST /links/shorten (Работает + custom_alias)
- GET /{short_code}
- DELETE /links/{short_code}
- PUT /links/{short_code}
- GET /links/{short_code}/stats

Включает:
- Кэширование Redis
- Custom alias
- Поиск по original_url
- expires_at
- Docker Compose (+ Healthchecks)
- База данных PostgreSQL

## Краткий год по POST|GET методам на примере на сайта Flowwow
- Первый post|get отвечает за статус страницы
- Второй post|get отвечает за генерацию короткой ссылки
- Третий post|get показывает статистику по ссылке с шага 2
- Редирект по сгенерированной ссылке отправит на исходный сайт
```bash
POST: curl http://localhost:8000/health
GET: {"status":"healthy"}

POST: curl -X POST "http://localhost:8000/links/shorten" \
    -H "Content-Type: application/json" \
    -d '{"original_url": "https://flowwow.com"}'
GET: {"short_url":"http://localhost:8000/[YOUR CODE]",
        "short_code":"[YOUR CODE]"}

POST: curl http://localhost:8000/links/[YOUR CODE]/stats
GET: {"original_url":"https://flowwow.com/",
        "created_at":"2026-03-02 17:40:53.072191+00:00", "clicks":0, "last_used_at":null,"expires_at":null}
```
